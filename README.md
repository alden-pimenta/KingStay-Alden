# King Stay – Luxury Hotel Listing Platform

## Overview
King Stay is a full-stack web application designed to provide a seamless platform for exploring and managing luxury hotel listings. It delivers a premium user experience with a clean and interactive interface, allowing users to browse, create, and manage property listings efficiently.

This project demonstrates real-world full-stack development using modern technologies such as Node.js, Express.js, MongoDB, and EJS.

---

## Features

### User Features
- User signup and login (authentication)
- Session management (persistent login)
- Browse all hotel listings
- View detailed property information

### Listing Management
- Create new property listings
- Edit existing listings
- Delete listings (restricted to owner)
- View listings in a structured UI

### Review System
- Add reviews to listings
- Delete reviews
- Reviews linked to specific listings

### Security
- Authentication using Passport.js
- Authorization (only owners can edit or delete)
- Password hashing and secure session handling
