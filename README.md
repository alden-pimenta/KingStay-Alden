👑 King Stay – Luxury Hotel Listing Platform

Overview
King Stay is a full-stack web application designed to provide a seamless platform for exploring and managing luxury
hotel listings. It offers users a premium experience to browse, create, and manage property listings with a clean and 
interactive interface.

This project demonstrates real-world implementation of full-stack development using modern technologies like Node.js,
Express.js, MongoDB, and EJS .

Features
  User Features
    User Signup & Login (Authentication)
    Session Management (Stay logged in)
    Browse all hotel listings
    View detailed property information

Listing Management
   Create new property listings
   Edit existing listings
   Delete listings (only by owner)
   View listings in structured UI

Review System
   Add reviews to listings
   Delete reviews
   Linked reviews with listings

Security

   Authentication using Passport.js
   Authorization (only owner can edit/delete)
   Password hashing and session handling



Tech Stack

Frontend
   HTML
   CSS
   JavaScript
   EJS (Embedded JavaScript Templates)

Backend
   Node.js
   Express.js

Database
 MongoDB


Other Tools & Packages

   Passport.js (Authentication)
   express-session
   connect-mongo
   method-override
   dotenv
   connect-flash

Future Improvements

   Payment integration 
   Advanced search filters 
   Real-time booking system 
   Better data validation 
   User ratings & recommendation system 

Conclusion

King Stay successfully demonstrates a complete full-stack web application with secure authentication, 
structured routing, and dynamic data handling. It serves as a strong foundation for building scalable real-world 
applications.
